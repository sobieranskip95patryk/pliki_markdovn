# Rozmowy z Bogiem

Minimalistyczna strona w formie osi czasu, gdzie każdy wpis to kolejna rozmowa z Bogiem.

- Centralna przestrzeń rozmowy (hero)
- Oś rozmów w dół (jak blog / komentarze)
- Formularz do dodawania nowych wpisów

## Uruchomienie

1. Otwórz `index.html` w przeglądarce  
2. Lub wrzuć repo na GitHub i włącz GitHub Pages
